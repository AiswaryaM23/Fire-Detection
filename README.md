# 🔥 Fire Detection

### Machine Learning Based Early Fire Detection System

Fire Detection is an intelligent surveillance system designed to identify fire incidents at an early stage using Computer Vision and Machine Learning techniques. The system analyzes video streams captured through CCTV cameras or webcams, extracts relevant visual features, and classifies the scene as fire or non-fire. By enabling rapid detection and automated alert generation, the system aims to minimize property damage, environmental loss, and potential threats to human life.

---

## 🧠 AI & Machine Learning Concept

* Computer Vision techniques are used to process video streams and extract image frames.
* Image preprocessing methods such as grayscale conversion, resizing, and feature extraction are applied to improve model performance.
* Support Vector Machine (SVM) and Convolutional Neural Network (CNN) models are trained to classify fire and non-fire images.
* Model performance is evaluated and compared based on classification accuracy.
* Automated alert generation is triggered when fire is detected.

---

## ✨ Key Features

* Real-time fire detection from CCTV or webcam video feeds.
* Fire and non-fire image classification using Machine Learning models.
* Comparative analysis of SVM and CNN algorithms.
* Automated emergency alert notification system.
* Early detection mechanism to reduce response time and potential damage.

## ▶️ How to Run the Application

* Clone the repository to your local machine.
* Install the required Python libraries and dependencies.
* Prepare the fire and non-fire image dataset.
* Train the machine learning model or load the pre-trained model.
* Configure the alert notification settings.
* Execute the application and provide a video stream from a webcam or CCTV feed.
* The system will analyze incoming frames and generate alerts when fire is detected.
